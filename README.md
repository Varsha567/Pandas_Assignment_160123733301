# Amazon Bestselling Books Analysis

This repository contains a comprehensive exploratory data analysis (EDA) and market study of the Amazon Bestselling Books dataset. The project focuses on understanding book popularity, pricing trends, and genre comparisons using Python and pandas.

---

## Dataset

- Filename: `bestsellers with categories.csv`  
- Description: Contains bestselling book information including Title, Author, User Rating, Reviews, Price, Year, and Genre.

---

## Project Overview

This analysis covers:

- Handling missing data with different imputation strategies  
- Hierarchical indexing to analyze trends across genres and years  
- Creating a Book Popularity Index combining user ratings and reviews  
- Pricing tier classification for budget, standard, and premium books  
- Identifying overpriced and underpriced books relative to genre averages  
- Genre-wise comparisons of ratings, prices, and review counts  
- Extracting high-performing books and analyzing author and genre distributions  
- Using boolean indexing, vectorized operations, and groupby methods for insights  
- Visualizations of price trends, popularity scores, and tier distributions
